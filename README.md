# snbt-translator
FTB quest의 snbt 파일을 온전히 보존하면서 en파일을 ko파일로 변환합니다.

## 사용법
en_us.snbt와 cli_translator.py를 같은 폴더에 두고 해당위치에서 터미널을 열어서 아래에 있는 해당 코드를 치시면 됩니다.

```bash
#의존성 설치(최초 1회만 실행)
pip install -r requirements.txt

# 번역기 실행
python cli_translator.py
```
